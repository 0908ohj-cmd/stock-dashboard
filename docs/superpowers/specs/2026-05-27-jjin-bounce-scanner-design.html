찐반등 스캐너 & RS 와치리스트

설계 문서 · 2026-05-27
지수가 하락하는 동안 이미 독자적 상승 흐름을 보이는 종목(저점 선행, 이평선 위 유지)이 시장 회복 시 가장 먼저, 가장 강하게 오른다. — 미너비니 주도주 원칙

1. 아키텍처

app.py
├── strategy/market_status.py   ← 신규: 지수 상태 감지
├── strategy/rs_correction.py   ← 신규: 조정 구간 RS 계산
├── ui/watchlist.py             ← 전면 개편: 새 정렬 기준
└── ui/intraday_overlay.py      ← 신규: 5분봉 오버레이 차트

데이터 흐름

지수 OHLCV
  → 찐반등 날 자동 감지 (market_status)
  → 조정 구간 확정 (21EMA 이탈일 ~ 찐반등 날)

종목 OHLCV (CSV 업로드)
  → 조정 구간 RS 계산 (rs_correction)
  → 와치리스트 정렬

종목 선택
  → 찐반등 날 5분봉 fetch
  → 오버레이 차트 (intraday_overlay)

2. 찐반등 감지 (strategy/market_status.py)

조기 신호 조건 (지수 일봉 기준)

#조건비고
1지수 종가 < EMA21조정 구간 전제
2당일 상승폭 ≥ ADR(14일)절대적 크기 확보
3직전 봉 음봉이고, 당일 양봉 바디 ≥ 직전 음봉 바디 × 70%전봉 커버
4거래량 ≥ 20일 평균 → ★ 보너스필수 아님

갭업 포함 — 뉴스 이벤트 갭업도 가격 조건 충족 시 찐반등으로 인정.

강도 등급

등급조건
★기본 조건 충족, 거래량 평균 미만
★★기본 조건 + 거래량 ≥ 20일 평균
★★★기본 조건 + 거래량 ≥ 20일 평균 × 1.2

FTD 확인 (보조 표시)

찐반등 이후 Day 1 저점 유지 상태에서 Day 4+ 에 +1.7% 이상 + 전일 대비 거래량 증가. 매수 의사결정에 사용하지 않고 상태 카드 보조 정보로만 표시.

반환 구조

get_market_status(index_df) -> dict
{
  "state":             "normal" | "correction" | "early_signal" | "ftd_confirmed",
  "correction_start":  date | None,   # 21EMA 이탈 시작일
  "jjin_date":         date | None,   # 찐반등 감지일
  "jjin_pct":          float,         # 당일 상승%
  "jjin_stars":        int,           # 1~3
  "ftd_date":          date | None,   # FTD 확인일 (보조)
}

3. 조정 구간 RS 계산 (strategy/rs_correction.py)

측정 구간: 지수 21EMA 이탈일 ~ 찐반등 날

지표

지표계산의미
excess_pct종목 수익률 − 지수 수익률 (구간 전체)조정 구간 초과수익
lead_days지수 저점 날짜 − 종목 저점 날짜양수 = 지수보다 먼저 저점
ma_score찐반등 날 기준 EMA10/21, SMA50/150/200 위 개수 (0~5)이평선 구조 강도
vol_ratio상승일 평균거래량 / 하락일 평균거래량 (구간)매수 우위 거래량
candle_ratio누적 양봉 바디합 / 음봉 바디합 (구간)양봉 질적 우위

정렬 우선순위

  1. lead_days 내림차순 — 지수보다 먼저 저점 찍은 종목
  2. excess_pct 내림차순 — 조정 구간 초과수익률
  3. ma_score 내림차순 — 찐반등 날 이평선 위 개수
  4. vol_ratio 내림차순 — 거래량 품질 (덤)

5. 찐반등 날 5분봉 오버레이 (ui/intraday_overlay.py)

⚠️ 데이터 제약: yfinance 5분봉은 최근 60일 이내만 제공. 초과 시 "5분봉 데이터 없음" 표시.

차트

정규화 누적수익률 오버레이 — 당일 시가 = 0% 기준으로 지수(회색 점선) + 종목(컬러 실선)을 겹쳐 표시. 지수 당일 고점 시각에 수직 점선 표시.

인트라데이 강도 지표

지표계산
지수 고점 이후 초과 상승%지수 고점 시각 이후 종목 수익률 − 지수 수익률
저점 방어 비교지수 저점 이탈 횟수 vs 종목 저점 이탈 횟수
고점 갱신 횟수 비교종목 5분봉 신고가 갱신 횟수 vs 지수 갱신 횟수
오후 버팀 비율종가/당일고점 vs 지수종가/지수당일고점
